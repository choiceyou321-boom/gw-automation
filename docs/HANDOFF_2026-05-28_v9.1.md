## 9. 글로우 PM과의 관계

- PM 백엔드 코드는 **양 리포에 동일 복사본** 존재 (`src/pm/fund_table/`)
- 데이터 DB (`fund_management.db`)는 **각각 별도** — 분기 의도적
- ENCRYPTION_KEY는 공유 (양쪽 .env에 동일 값)
- users.db 도 양쪽 복사본이 같이 출발 (이후 개별 진화)
- 글로우 PM의 UI 화면(/login + 12 라우트)은 본 리포에 없음
