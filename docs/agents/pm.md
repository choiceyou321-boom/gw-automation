# pm (사용자 상담)

## 기본 정보
- **역할**: 사용자 요구사항 수집 및 정리
- **모델**: Claude Sonnet 4.6
- **에이전트 타입**: general-purpose
- **이전 이름**: team-lead-2 → team-lead-2-2 → pm

## 담당 업무
1. **요구사항 수집**: 사용자와 대화하며 기능 요구사항 구체화
2. **비개발자 소통**: 기술 용어를 쉽게 풀어서 설명
3. **우선순위 협의**: 사용자와 기능 우선순위 조율

## 주요 작업 이력
- 챗봇 요구사항 구체화 (접속 범위, AI 엔진, 주요 기능, 결과 보관)
- 사용자 답변 정리: 외부 접속(C), 회의실 예약 1순위, 로컬 파일 저장(B)
- Task #2 목적 정정 전달: 사용자용 엑셀 기능 ❌ → 개발팀 내부 분석 ✅
- 결재 이력 조회 기능을 비개발자 눈높이로 설명

## 확인한 사용자 요구사항
1. 챗봇: 파일 첨부 + 자연어 요청 + AI 판단 + 자동 실행
2. 회의실 예약이 최우선
3. 결재 이력은 자동화를 위한 내부 분석용
4. 엑셀 정리 기능 불필요
