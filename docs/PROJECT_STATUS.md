1. **OBTDataGrid = 더존 자체 canvas 그리드** — `window.gridView` 전역 변수 없음, React fiber 경로로 접근
2. **접근 경로**: `.OBTDataGrid_grid__22Vfl` → `__reactFiber` → depth 3 → `stateNode.state.interface`
3. **OBTGrid (canvas 기반)**: DOM 행 요소 없음 → 모달 제목 기준 좌표 더블클릭만 유효
4. **OBTDialog2 overlay**: `dimClicker` 레이어가 뒤쪽 요소 차단 → JS로 dialog 컨테이너 내부 요소 직접 탐색
5. **GW 폼 2종**: 인라인(지출결의서, 보관 버튼 없음) vs 팝업(거래처등록, 보관 있음)
6. **wehago-sign**: `Base64(HMAC-SHA256(signKey, oAuthToken + transactionId + timestamp + pathname))`
