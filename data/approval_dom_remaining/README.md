## 핵심 발견사항

### 1. 연장근무/외근신청 — 근태관리 모듈 (전자결재 아님)
- 결재작성에서 검색하면 찾을 수 있지만, 실제 신청 화면은 **근태관리 > 근태신청** 모듈로 이동
- URL: `/#/...` 근태 관련 URL로 전환
- 신청 방식: 별도 근태 API 사용 (wehago-sign 기반 추정)
- **자동화 접근법**: 기존 전자결재 Playwright 방식 불가, 근태 API 별도 분석 필요

### 2. 선급금요청/선급금정산 — 지출결의서 변형
- 전자결재 결재작성에서 열리며 `지출결의서작성` 화면과 동일한 UI
- 왼쪽 양식 목록에서 `[본사]선급금 요청서` / `[본사]선급금 정산서` 선택 시 해당 양식으로 전환
- **자동화 접근법**: 기존 지출결의서 자동화 코드 재사용 + 양식 선택 로직만 추가

### 3. 증빙발행/사내추천비 — formId 추가 탐색 필요
- 결재작성 검색에서 발견됨 (양식명 확인)
- Enter 후 팝업이 아닌 인라인으로 열릴 것으로 추정
- formId를 URL에서 확인하려면 직접 양식 열기 후 URL 캡처 필요
