# approval-dev (결재 자동화)

## 기본 정보
- **역할**: 결재 문서 분석 및 자동화 개발
- **모델**: Claude Sonnet 4.6
- **에이전트 타입**: general-purpose

## 담당 업무
1. **결재 양식 분석**: 기존 결재 이력에서 양식/패턴 파악 (내부 분석용)
2. **결재 자동 작성**: 분석된 패턴 기반 결재 문서 자동 생성 (향후)
3. **그룹웨어 연동**: Playwright로 결재 페이지 자동화

## 주요 작업 이력
- `src/approval/history.py` 코드 작성 완료
- 결재 이력 조회 기능 구현 (네트워크 인터셉트 포함)
- 실제 그룹웨어 테스트 대기 중

## 담당 파일
- `src/approval/history.py` - 결재 이력 조회/분석
- (향후) `src/approval/submit.py` - 결재 자동 작성

## 참고 사항
- Task #2 (결재 이력 조회)는 사용자용 기능이 아닌 **개발팀 내부 분석**용
- 분석 결과는 향후 Task #5 (결재 자동 작성)의 기반 데이터로 활용
