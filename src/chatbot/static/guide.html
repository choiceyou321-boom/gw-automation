GW 자동화 비서 사용 가이드

글로우서울 그룹웨어 업무를 자연어 AI로 자동화합니다. 회의실 예약부터 전자결재, 프로젝트 관리까지.

📋 전체 기능 개요

🏢
회의실 예약
예약 · 취소 · 조회 · 빈 시간 검색. 자연어로 "내일 3시 2번 회의실 예약해줘"
📝
전자결재 자동화
지출결의서, 거래처등록 등 5종. AI가 양식을 자동 작성하고 임시보관합니다.
📄
계약서 생성
자재납품 · 공사 계약서 자동 생성. 엑셀 첨부로 일괄 생성도 가능.
💰
프로젝트 관리
수금 · 하도급 · 예산 · 이체내역을 한눈에. GW 데이터 자동 동기화.
🤖
AI 인사이트
Gemini AI가 프로젝트별 리스크 · 자금흐름 · 실행 권고를 분석합니다.
🎤
음성 인식
음성 파일 첨부 시 자동으로 텍스트 변환 후 명령 실행.

🔑 로그인 / 회원가입

웹 브라우저로 접속
https://unchidingly-hybridizable-maranda.ngrok-free.dev 에 접속합니다.
회원가입 (최초 1회)
[회원가입] 탭에서 이름, 직급, 그룹웨어 아이디/비밀번호를 입력합니다. 비밀번호는 암호화 저장됩니다.
로그인
[로그인] 탭에서 그룹웨어 아이디와 비밀번호로 접속합니다. 24시간 동안 로그인이 유지됩니다.

💬 웹 챗봇

채팅창에 자연스러운 대화를 입력하면 AI가 알아서 처리합니다. 명령어 규칙 없이 편하게 말하세요.

화면 구성

영역기능
왼쪽 사이드바[+ 새 대화] 새 주제 시작 / 빠른 실행 버튼 (회의실, 지출결의서 등) / 이전 대화 목록
오른쪽 상단🗑️ 대화 초기화 / [로그아웃]
채팅 입력창텍스트 입력 또는 파일 첨부 (드래그앤드롭 지원)

파일 첨부

영수증, 사업자등록증, 세금계산서 등을 드래그앤드롭으로 첨부하면 AI가 자동 인식합니다.

💡
지원 형식: JPG, PNG, PDF (파일당 최대 10MB). 사업자등록증 첨부 시 상호·대표자·사업자번호 자동 추출.

🏢 회의실 예약

1~5번 회의실을 자연어로 예약 · 취소 · 조회할 수 있습니다.

예시 문장
  • 내일 오후 3시에 2번 회의실 예약해줘. 참석자는 홍길동, 김철수야.
  • 오늘 1시부터 2시간 동안 쓸 수 있는 빈 회의실 찾아줘.
  • 내일 잡아둔 내 회의실 예약 취소할래.
  • 내 예약 목록 보여줘.

📝 전자결재 자동 작성

AI가 양식을 자동으로 채워서 임시보관문서 폴더에 저장합니다. 확인 후 GW에서 직접 [상신] 버튼을 눌러야 결재가 진행됩니다.

양식상태설명

거래처등록사용 가능사업자등록증 + 통장사본 첨부 후 요청.
선급금요청준비 중코드 완성, GW 검증 필요.
연장근무예정근태관리 모듈 연동 예정.
외근신청예정근태관리 모듈 연동 예정.
예시 문장
  • 이번 달 야근 식대 2만원 지출결의서 올려줘.
  • 사업자등록증이랑 통장사본 첨부했어. 거래처등록 해줘.

결재선 설정

챗봇이나 텔레그램에서 결재선을 커스텀할 수 있습니다.

결재선 설정 예시
  • 내 결재선을 검토:신동관, 승인:최기영으로 설정해줘
⚠️
보관 모드: AI는 서류를 임시보관만 합니다. 내용 확인 후 GW에서 직접 [상신]을 눌러야 결재가 진행됩니다.

❓ 자주 묻는 질문

AI가 결재를 바로 올려버리나요?

아닙니다. AI는 임시보관까지만 수행합니다. 사용자가 GW에서 직접 확인 후 [상신]을 눌러야 합니다.

비밀번호가 유출되지 않나요?

비밀번호는 Fernet 암호화되어 로컬 서버에 저장됩니다. AI 엔진으로는 전송되지 않습니다. 텔레그램에서는 비밀번호 메시지가 자동 삭제됩니다.

AI가 말을 잘 못 알아듣는 것 같아요

대화가 길어지면 문맥 혼동이 생길 수 있습니다. [+ 새 대화] 또는 텔레그램 /clear로 초기화하세요.

프로젝트 관리 페이지에서 데이터가 안 보여요

좌측 사이드바에서 프로젝트를 선택해야 탭 내용이 표시됩니다.

GW 동기화가 실패했어요

GW 세션이 만료되었을 수 있습니다. 챗봇에서 다시 로그인 후 동기화를 재시도하세요. 3단계 중 일부가 실패해도 나머지는 진행됩니다.

예실대비 데이터에 전기 내용이 안 보여요

사업코드가 올바르게 입력되어 있는지 확인하세요. 사업별 크롤러가 시작 연도부터 현재까지 자동 조회합니다.

GW 데이터가 자동 업데이트되나요?

네. 매일 아침 8시에 자동 동기화됩니다. 급할 때는 [전체 GW 동기화]로 수동 실행도 가능합니다.
