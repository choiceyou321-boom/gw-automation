# 🤖 글로우서울 그룹웨어(WEHAGO) 자동화 비서 - 사용자 매뉴얼

안녕하세요! 글로우서울 그룹웨어 업무를 자동화해주는 **AI 비서** 이용 안내서입니다.
사용자는 **텔레그램 메신저**와 **웹 챗봇** 두 가지 채널 중 편한 곳을 선택하여 이용할 수 있습니다. 
현재 **회의실 예약 관리** 및 **전자결재(지출결의서, 거래처등록 등) 자동 작성** 기능을 지원합니다.

---

## � 1. 텔레그램 봇 전용 가이드

언제 어디서나 스마트폰을 통해 그룹웨어 업무를 빠르게 처리하고 싶을 때 추천합니다.

### 1.1. 접속 및 초기 설정
1. **봇 검색 및 시작**: 텔레그램 앱 실행 후 관리자에게 안내받은 봇 아이디를 검색합니다. 대화방 하단의 **[시작]**을 누르거나 `/start` 를 입력합니다.
2. **회원가입**: 처음 사용 시 아래 형식으로 가입을 진행합니다.
   - 입력 형식: `/register [그룹웨어아이디] [그룹웨어비밀번호] [이름] [직급]`
   - 입력 예시: `/register tgjeon mypass123 전태규 대리`
3. **로그인**: 가입을 마친 후 아래 형식으로 로그인합니다.
   - 입력 형식: `/login [그룹웨어아이디] [그룹웨어비밀번호]`
   - *참고: 전송하신 비밀번호 로그는 보안을 위해 서버 저장 전 암호화되며, 대화창에서도 자동으로 삭제처리 됩니다.*

### 1.2. 텔레그램 기본 명령어
- `/start`: 봇 시작 및 기본 안내 확인
- `/register`: 신규 회원가입
- `/login`: 그룹웨어 연동 로그인
- `/clear`: 현재까지의 **대화 내용(기억)을 초기화**하고 새로운 주제로 대화를 시작합니다. (AI가 이전 문맥을 혼동할 때 유용)

### 1.3. 텔레그램 사용 방법 및 예시
텔레그램의 대화창에 명령어 없이 **자연스러운 대화**로 업무를 지시하세요.

* **회의실 예약/조회**
  * "내일 오후 3시에 2번 회의실 예약해줘. 참석자는 홍길동, 김철수야."
  * "오늘 1시부터 2시간 동안 쓸 수 있는 빈 회의실 찾아줘."
  * "내일 잡아둔 내 회의실 예약 취소할래."
* **전자결재 자동 작성 (지출결의서, 거래처등록 등)**
  * "이번 달 야근 식대 2만원 지출결의서 올려줘."
  * **파일 첨부하기**: 영수증 사진(JPG, PNG 등)이나 사업자등록증(PDF)을 보내야 할 경우, 텔레그램의 📎(클립) 버튼을 눌러 파일을 전송한 뒤 어떤 결재인지 텍스트로 설명해주세요. (최대 10MB)
* **메일 요약** (세션 VI 신규 완료)
  * "메일 요약해줘." — 안 읽은 메일 목록과 내용을 AI가 요약해서 알려줍니다.
  * "새 메일 있어?" — 미읽은 메일 현황을 빠르게 확인할 수 있습니다.
  * *참고: 본인이 수신인(To)으로 지정된 메일만 요약됩니다. 참조(CC)로만 받은 메일은 제외됩니다.*

## 💻 2. 웹 서버 챗봇 전용 가이드

PC 환경에서 큰 화면으로 여러 대화 세션을 관리하며 파일 첨부를 편리하게 하고 싶을 때 추천합니다.

### 2.1. 접속 및 초기 설정
1. **접속하기**: 사내망 또는 관리자가 안내한 주소(기본 `http://localhost:51749`)를 웹 브라우저 주소창에 입력하여 접속합니다.
2. **회원가입**: 화면 중앙의 **[회원가입]** 탭을 클릭한 뒤, 본인의 이름, 직급, 그룹웨어 아이디 및 비밀번호를 입력해 가입합니다. 가입이 완료되면 자동으로 로그인 탭으로 이동합니다.
3. **로그인**: **[로그인]** 탭에서 그룹웨어 아이디와 비밀번호를 입력하여 접속합니다. (최초 로그인 이후에는 브라우저를 닫아도 일정 기간 로그인이 유지됩니다.)

### 2.2. 웹 챗봇 화면 구성 (UI)
- **왼쪽 사이드바 (메뉴)**: 
  - **[+ 새 대화]**: 새로운 주제로 대화를 시작합니다. 버튼을 누르면 이전 대화 내용은 사이드바 목록에 저장됩니다.
  - **빠른 실행 버튼**: 일일이 타이핑할 필요 없이 `회의실 예약`, `지출결의서`, `거래처등록` 등의 버튼을 누르면 요청이 자동 전송됩니다.
  - **이전 대화 목록**: 과거에 AI와 나눴던 대화 내역(세션)을 클릭해 다시 이어서 작업할 수 있습니다. ❌ 버튼을 눌러 개별 대화를 삭제할 수도 있습니다.
- **오른쪽 상단 메뉴**:
  - **휴지통(🗑️) 아이콘**: 텔레그램의 `/clear`와 동일하게 현재 대화방의 내용을 즉시 초기화합니다.
  - **[로그아웃]**: 현재 기기에서 안전하게 로그아웃합니다.
  - *(관리자 전용)* **[관리]**: 구성원의 권한을 관리할 수 있는 관리자 전용 페이지로 이동합니다.

### 2.3. 웹 챗봇 사용 방법 및 예시
명령어 규칙 없이 입력창에 **자연스러운 대화**를 입력하거나, 빠른 실행 버튼을 클릭합니다.

* **직관적인 파일 첨부 (드래그 앤 드롭)**
  * 영수증, 통장사본, 사업자등록증 등의 파일(JPG, PNG, PDF 지원, 파일당 최대 10MB)을 **PC 폴더에서 드래그하여 채팅창 위로 끌어다 놓으면** 즉시 첨부됩니다.
  * 입력창 좌측의 📎(클립) 아이콘을 클릭해 파일을 직접 선택할 수도 있습니다.
  * 여러 장의 사진을 한 번에 올려서 "이 영수증들 다 합쳐서 식대 지출결의서 써줘"라고 요청할 수 있습니다.
* **메일 요약** (신규)
  * "메일 요약해줘." — 안 읽은 메일 목록과 내용을 AI가 요약해서 알려줍니다.
  * "새 메일 있어?" — 미읽은 메일 현황을 빠르게 확인할 수 있습니다.
  * *참고: 본인이 수신인(To)으로 지정된 메일만 요약됩니다. 참조(CC)로만 받은 메일은 제외됩니다.*
* **전자결재 최종 확인**
  * AI가 파일과 대화를 분석하여 그룹웨어 상에 서류를 임시 작성(보관)합니다.
  * 작성된 내용을 AI가 요약해 주면, 챗봇상에서 **"맞아, 진행해"** 라고 승인해주시면 모든 절차가 완료됩니다!

## 💡 3. 공통 유의사항 및 FAQ

- **Q. AI가 예약이나 결재를 최종 승인(상신)까지 바로 해버리나요?**
  - **아닙니다.** 가장 중요한 전자결재의 경우, AI는 문서를 자동으로 작성하여 사용자의 그룹웨어 **'임시보관문서'** 폴더에 저장(보관 모드)해두는 것까지만 수행합니다. 사용자가 그룹웨어에 접속해 작성된 내용을 최종 확인한 후 직접 [상신] 버튼을 눌러야 결재가 진행됩니다. (오작동 방지)
- **Q. AI가 말을 잘 못 알아듣는 것 같아요.**
  - 대화가 너무 길어지면 AI가 이전 문맥과 현재 지시를 혼동할 수 있습니다. 주제가 바뀔 때는 텔레그램은 `/clear` 입력, 웹 버전은 **[+ 새 대화]** 버튼이나 우측 상단 **휴지통(🗑️)** 버튼을 눌러 대화를 깔끔하게 비우고 새로 시작하는 것이 좋습니다.
- **Q. 비밀번호를 채팅창에 치는 게 불안해요.**
  - 입력하신 그룹웨어 비밀번호는 AI 엔진(Gemini 등)으로 절대 전송되지 않으며, 사내 로컬 서버 데이터베이스에 안전하게 암호화(Fernet 알고리즘)되어 저장됩니다.
