# 자동화 Work 프로젝트

## 프로젝트 개요
자동화 작업을 위한 프로젝트입니다.

## Agent Teams 가이드

### 팀 생성 방법
- 세션 내에서 "3명의 에이전트 팀을 만들어서 이 작업을 분담해줘" 와 같이 요청
- 각 팀원은 독립적으로 작업하며 공유 태스크 리스트로 소통

### 팀 단축키
- `Shift+Down`: 팀원 간 전환
- `Ctrl+T`: 태스크 리스트 보기

### 코드 스타일
- 한국어 주석 사용
- 명확한 변수명 사용

### 작업 규칙
- 테스트 후 커밋
- 동일 파일 동시 편집 금지
- Agent 작업은 항상 백그라운드(`run_in_background: true`)에서 진행
