# 🤖 글로우서울 그룹웨어 자동화 비서 - 텔레그램 이용 가이드

안녕하세요! 글로우서울 그룹웨어(WEHAGO) 업무를 스마트폰에서 텔레그램 메신저 하나로 간편하게 처리할 수 있는 **AI 비서** 이용 안내서입니다.
현재 **회의실 예약/조회/취소** 및 **지출결의서 임시작성** 기능을 지원합니다.

---

## 🚀 1. 시작하기 (최초 1회 설정)

텔레그램 봇을 처음 사용하실 때 계정 등록 및 연동을 진행해 주시면 됩니다.

### 단계별 접속 방법
1. **텔레그램 앱 실행**: 스마트폰이나 PC에서 텔레그램 앱을 켭니다.
2. **봇 검색**: 상단 돋보기(검색) 아이콘을 누르고 관리자에게 안내받은 **봇 아이디**를 검색합니다. (예: 봇 링크 클릭)
3. **대화 시작**: 봇과의 대화방 하단에 있는 **[시작]** 버튼을 누르거나, 채팅창에 `/start`를 입력합니다.
4. **회원가입**:
   아직 가입하지 않으셨다면, 아래 양식으로 **그룹웨어 아이디, 비밀번호, 이름**을 입력해 가입합니다.
   > 입력 예시: `/register tgjeon mypass123 전태규 대리`
5. **로그인 (연동)**:
   가입 후, 대화창에 아래와 같은 양식으로 **그룹웨어 아이디**와 **비밀번호**를 입력하고 전송합니다.
   > 입력 예시: `/login tgjeon mypass123`
   
   *(참고: 입력하신 비밀번호는 보안상 암호화되어 안전하게 보관되며, 입력 직후 채팅방에서 메시지가 자동으로 삭제됩니다.)*
6. 봇이 `✅ 로그인 성공! 환영합니다, OOO님.` 이라고 답변하면 준비 완료입니다! 🎉

## 🗣️ 2. 어떻게 말하면 되나요?

명령어를 외울 필요 없이, 평소 동료에게 부탁하듯 **자연스러운 대화(언어)**로 말씀해 주시면 AI가 알아서 이해하고 처리합니다.

### 🏢 [회의실 예약 및 조회]

**1. 빈 회의실 찾기**
- "오늘 오후 2시부터 1시간 쓸 수 있는 빈 회의실 찾아줘"
- "내일 3번 회의실 비어있어?"
- "남는 회의실 어디야?"

**2. 회의실 예약하기**
- "내일 오전 10시 반에 3번 회의실 팀미팅 잡아줘. 참석자는 홍길동, 김철수야."
- "모레 오후 2시에 1시간짜리 회의실 아무데나 잡아줘."
*(AI가 정보가 부족하다고 판단하면, "언제 할까요?" 하고 친절하게 다시 물어봅니다.)*

**3. 내 예약 확인 및 취소하기**
- "오늘 내 예약 현황 보여줘."
- "내일 오후 3시 예약 취소하고 싶어."
- "방금 잡은 예약 취소해줘."

---

### 💸 [전자결재 - 지출결의서 작성] (Beta)

번거로운 지출결의서 양식을 텔레그램에서 말 한마디로 자동 작성(결재상신)할 수 있습니다.

**사용 예시**
- "이번 달 야근 식대 3만원 지출결의서 써줘."
- "3월 교통비 15000원 결재 올려줘. 프로젝트는 고수동굴이야."

**진행 순서**
1. 봇에게 지출 내용을 말합니다.
2. 봇이 작성할 내용(제목, 금액, 날짜 등)을 깔끔하게 정리해서 먼저 보여주며 **맞는지 확인**을 요청합니다.
3. 만족스러우면 **"맞아, 작성해줘"** 또는 **"진행해"**라고 답변합니다.
4. 🤖 봇이 그룹웨어에 접속하여 지출결의서 양식을 자동으로 채우고 '결재상신' 처리를 완료합니다!

## 💡 3. 유용한 팁 (FAQ)

- **Q. 다시 로그인해야 하나요?**
  A. 아닙니다! 최초 1회만 `/login`을 하시면 이후에는 계속 로그인 상태가 유지됩니다.
- **Q. 말을 좀 대충 해도 되나요?**
  A. 네! "내일 2시 회의 잡아" 처럼 짧게 말씀하셔도 똑똑한 AI가 문맥을 파악합니다. 시간(오후 2시 → 14:00)이나 날짜(내일, 이번주 금요일 등)도 찰떡같이 알아듣습니다.
- **Q. 대화 내용을 지우고 싶을 때는?**
  A. `/clear`를 입력하면 현재 대화 기록이 초기화됩니다. 로그인 상태는 그대로 유지됩니다.
- **Q. 영수증이나 PDF를 첨부할 수 있나요?**
  A. 네! 이미지(JPG, PNG)나 PDF 파일을 채팅창에 올리면서 메시지를 함께 보내면 봇이 내용을 참고하여 처리합니다.
- **Q. 에러가 나거나 로봇이 헤맬 때는요?**
  A. 아직 봇이 학습 중인 단계(Beta)이므로, 너무 복잡한 요청에는 가끔 엉뚱한 대답을 할 수 있습니다. 그럴 땐 내용을 조금 더 구체적으로(예: 날짜, 시간, 회의실명 명시) 다시 말씀해 주시면 좋습니다. 중대한 오류는 관리자(전태규 대리)에게 문의해 주세요.

## 4. 명령어 전체 목록

| 명령어 | 기능 |
|--------|------|
| `/start` | 봇 시작 및 사용 안내 확인 |
| `/register {id} {pw} {이름}` | 시스템 회원가입 |
| `/login {id} {pw}` | 그룹웨어 계정 연동 (로그인) |
| `/clear` | 현재 대화 기록 초기화 (로그인 상태 유지) |

*(참고: `/newchat` 명령어는 세션 IV에서 제거되었습니다. 대신 `/clear`를 사용하세요.)*
